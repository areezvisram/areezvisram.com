# My Personal Website
### http://areezvisram.com
#### Description: 
* Built with GatsbyJS and React. Styling done with SCSS
* Fully responsive for all devices, desktop and mobile
* Built to show my personal portfolio of software skills, experience and projects
### Installation:
1. Install the Gatsby CLI

   ```sh
   npm install -g gatsby-cli
   ```

2. Install and use the correct version of Node using [NVM](https://github.com/nvm-sh/nvm)

   ```sh
   nvm install
   ```

3. Install dependencies

   ```sh
   yarn
   ```

4. Start the development server

   ```sh
   gatsby develop
   ```
